Mashora is a suite of web based open source business apps.

The main Mashora Apps include an [Open Source CRM](https://www.mashora.com/page/crm),
[Website Builder](https://www.mashora.com/app/website),
[eCommerce](https://www.mashora.com/app/ecommerce),
[Warehouse Management](https://www.mashora.com/app/inventory),
[Project Management](https://www.mashora.com/app/project),
[Billing &amp; Accounting](https://www.mashora.com/app/accounting),
[Point of Sale](https://www.mashora.com/app/point-of-sale-shop),
[Human Resources](https://www.mashora.com/app/employees),
[Marketing](https://www.mashora.com/app/social-marketing),
[Manufacturing](https://www.mashora.com/app/manufacturing),
[...](https://www.mashora.com/)

Mashora Apps can be used as stand-alone applications, but they also integrate seamlessly so you get
a full-featured [Open Source ERP](https://www.mashora.com) when you install several Apps.

## Getting started with Mashora

For a standard installation please follow the [Setup instructions](https://www.mashora.com/documentation/master/administration/install/install.html)
from the documentation.

To learn the software, we recommend the [Mashora eLearning](https://www.mashora.com/slides),
or [Scale-up, the business game](https://www.mashora.com/page/scale-up-business-game).
Developers can start with [the developer tutorials](https://www.mashora.com/documentation/master/developer/howtos.html).

## Security

If you believe you have found a security issue, check our [Responsible Disclosure page](https://www.mashora.com/security-report)
for details and get in touch with us via email.
