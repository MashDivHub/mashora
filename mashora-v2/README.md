# Mashora V2

This folder is the new organized ERP workspace.

## Structure

- `frontend`
  - React + Vite + shadcn-style UI for the ERP
  - Replaces the legacy ERP frontend over time
- `backend`
  - Existing Mashora ERP server, addons, packaging, and runtime files
  - Intended to hold backend logic only while frontend work moves into `frontend`

## Run V2

```powershell
cd C:\xampp\htdocs\mashora\mashora-v2
.\start-v2.ps1
```

## Direct Paths

- Frontend: `C:\xampp\htdocs\mashora\mashora-v2\frontend`
- Backend: `C:\xampp\htdocs\mashora\mashora-v2\backend`
